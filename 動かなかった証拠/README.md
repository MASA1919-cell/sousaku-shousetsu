# 動かなかった証拠

### 新人弁護士が十年越しの再審請求で辿り着いた、黙殺された最後の証言

## あらすじ

司法修習中に見た、たった一件の再審法廷に心を動かされ、大手事務所の内定をすべて断って、冤罪弁護専門の小さな事務所に入所した久我山楓(くがやま かえで)、二十六歳。任されたのは、十年前、傷害致死罪で有罪が確定し、今も服役中の本宮悠(もとみや ゆう)の再審請求だった。無実を訴え続けて誰にも信じてもらえず、心を閉ざした本宮。楓は、当時「証拠能力に乏しい」として不採用になった目撃証言や、実況見分調書に残る不自然な時間の空白に着目し、少しずつ、事件の真相へと迫っていく。十年前の証言を悔いる元目撃者・戸田、事件現場に居合わせながら沈黙してきた河野瑛太、そして、組織の圧力に屈して真実を隠し続けてきた元刑事・桐山。それぞれが抱えてきた十年分の後悔と向き合いながら、楓と師匠の不破弁護士は、動かなかった証拠を、一つずつ、掘り起こしていく。骨太な法廷ミステリ、全10話。

## 登場人物

- **久我山楓(くがやま かえで)** — 主人公・語り手。二十六歳の新人弁護士。冤罪弁護に強い使命感を持ち、本宮悠の再審請求を担当する
- **不破** — 楓の師匠にあたる、冤罪弁護専門の弁護士。三十年前、無実を訴えながら救えなかった依頼人の自死を経験し、それ以来、冤罪弁護に人生を捧げている
- **本宮悠(もとみや ゆう)** — 十年前、十九歳の時に傷害致死罪で有罪判決を受け、服役中の依頼人。無実を訴え続けたが誰にも信じてもらえず、心を閉ざしている
- **戸田喜一** — 事件当時の目撃者の一人。証言が不採用になった経緯に、十年間、後悔を抱え続けてきた
- **河野瑛太** — 事件当時、現場近くにいた、もう一人の目撃者。決定的な写真を、大切に保管していた
- **桐山** — 当時の捜査を指揮した元刑事。組織の圧力により、捜査の不備を隠していたが、最終的に真実を証言する決意をする

## 制作メモ

- 過去9作に続く10作目。ミステリ・サスペンスジャンルは、遺品整理士が主人公の第3作『さよならの部屋』以来だが、本作は個人の哀しみをめぐる連作ミステリではなく、法廷を舞台にした社会派の冤罪サスペンスとして、まったく異なる切り口で描いた
- 主人公像は「強い意志を持つ挑戦者」。第9作『風を読む仕事』に続き、目標に向かって困難を乗り越えていくタイプを採用し、正義感を軸に据えた
- メイン軸は謎・使命の解明。一話ごとに新しい証言者や手がかりを積み重ね、十年前の事件の真相へと収束させる構成にした
- トーン指定は「シリアス・骨太」。結末は、無罪判決という到達点を描きつつも、「失われた十年は戻らない」という現実を直視し、単純なハッピーエンドで終わらせない着地を選んだ
