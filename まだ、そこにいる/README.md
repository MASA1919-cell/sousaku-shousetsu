# まだ、そこにいる

### 特殊清掃員がたどり着く、二十年前から動けずにいる想いの物語

## あらすじ

就職活動に失敗し、藁にもすがる思いで内定を受けた柊木環(ひいらぎ たまき)、二十四歳。入社してわかったのは、「四季社」が遺品整理と特殊清掃を専門とする会社だということだった。ベテラン清掃員の白澤宗一(しらさわ そういち)に指導されながら現場に立つうち、環は、動かないはずの写真立てが元の位置に戻る現象、聞こえるはずのない子供の足音など、説明のつかない出来事に次々と遭遇する。四季社の本当の目的は、故人の未練が物や場所に染み付く「残留」という現象を鎮めること――白澤は、二十年前、故郷の集落の因習「山送り」で消えた妹・灯里(あかり)の行方を、この仕事を通じてずっと探し続けていた。環自身も幼くして弟を事故で亡くした過去を抱え、ある現場で自身の未練と残留が共鳴し、危険な状態に陥る。過去の依頼記録に潜む共通点から、二人は二十年前の真実へとたどり着いていく。シリアスで骨太な、喪失と鎮魂の物語、全10話。

## 登場人物

- **柊木環(ひいらぎ たまき)** — 主人公・語り手。二十四歳。就職難から特殊清掃・遺品整理会社「四季社」に入社する。十歳の時、川遊びの事故で弟の蒼(あおい)を亡くした過去を持ち、その未練が、仕事中の怪異と共鳴してしまう
- **白澤宗一(しらさわ そういち)** — 四季社のベテラン清掃員、四十代。環の指導役。二十年前、故郷の集落の因習「山送り」の儀式の晩に、妹の灯里が行方不明になって以来、残留を鎮める仕事を通じて彼女の行方を探し続けてきた
- **白澤灯里(しらさわ あかり)** — 白澤の妹。二十年前、十歳の時に「山送り」の儀式で祠へ向かい、消息を絶った。物語の核心にある、二十年間解けなかった謎の中心人物
- **柊木蒼(ひいらぎ あおい)** — 環の弟。十四年前、幼くして川の事故で亡くなった。環の未練の源であり、灯里との対話を通じて、環自身も彼との折り合いをつけていく

## 制作メモ

- 過去5作(異世界ファンタジー・人外バトル・ミステリ・月面科学空想・現代青春群像劇)に続き、ホラー・怪異ジャンルに初挑戦。トーン指定「シリアス・骨太」を踏まえ、過去作の中で最も重いテーマ(因習による子供の死、二十年間の罪悪感)を正面から扱った
- メイン軸は謎・使命の解明。「特殊清掃」という職業設定を活かし、一話ごとに小さな怪異(写真立て・足音・鏡)を積み重ねてから、白澤の妹の失踪という縦軸に収束させる構成にした
- 主人公の「普通の人が非日常に巻き込まれる」を、就職難から流れ着いた新卒社員という形で描き、「謎・使命の解明」を、白澤の二十年越しの探索という縦糸に託した。結末はハッピーエンドではなく、鎮魂と、それぞれの喪失との静かな和解として着地させた
- 執筆時の反省を踏まえ、1話書き終えるごとにその場で文字数を計測し2,800字を切ったら即座に増補する方式で進めたが、それでも全話で複数回の増補が必要だった。今回は他ジャンルよりテンポの速い会話中心の構成のため、全体字数は過去作よりやや少なめ(約24,300字)
