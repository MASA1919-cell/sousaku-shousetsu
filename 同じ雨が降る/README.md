# 同じ雨が降る

### 十月十七日、午後六時四十一分

## あらすじ

潮見市のガス保安点検員・相原透は、どこにでもいる三十二歳だった。夢もなく、恨みもなく、ただ真面目に町の配管を見て回るだけの男だった。その日、十月十七日の午後六時四十一分、駅前の地下街で爆発が起きた。二十七人が死んだ。透もその一人だった。

目を覚ますと、十月十七日の午前五時三十分だった。

逃げても、通報しても、ガス漏れを止めても、六時四十一分になれば町は焼ける。誰かを助ければ、代わりに別の誰かが死ぬ。数だけは決して動かない。繰り返しの底で、透はもう一人の「覚えている男」に出会う。地下街の設備管理人・田巻悟。彼はすでに何百回も同じ日を歩き、そのたびに、助かる者と死ぬ者の名前を選び直してきた。

数は動かない。動かせるのは名前だけだ——そう言い切る男と、全部を救いたいと言う凡人。二人は地下街の底に封じられた「第七計器室」へ降りていく。この日を終わらせる方法は一つだけあった。ただし、それには代価が要った。

## 登場人物

- **相原 透(あいはら とおる)** — 三十二歳。潮見ガス保安部の点検員。特別な才も志もない普通の男。武器はしつこさと、真っ当さだけ。
- **田巻 悟(たまき さとる)** — 五十一歳。潮見中央マートの設備管理人。元・市の下水道技師。透よりはるかに長く同じ日を繰り返している。娘のために。
- **田巻 美冬(たまき みふゆ)** — 十七歳。高校三年。爆発当日、地下街の書店で夕方から働いている。父とは五年、まともに口をきいていない。
- **楠木 澪(くすのき みお)** — 二十九歳。地下街の喫茶「みなと」店主。透が毎周回いちばん先に話す相手。町の手ざわりそのもの。
- **矢代 幾造(やしろ いくぞう)** — 八十四歳。地下街の靴修理屋。地下街ができる前の、この地下を知っている最後の一人。
- **阿部(あべ)** — 透の直属の係長。誠実だが、報告書と手順の外には出られない現実の壁。
- **相原 佳世(あいはら かよ)/ ひなた** — 透の姉と、小学五年の姪。透にとって「二十七」の外側にいてほしかった二人。

## 制作メモ

- タイムループもの(空想科学) × 巻き込まれ型の普通の人 × 一本の長い縦軸 × シリアス骨太(余韻の残る結末)。
- 作中の法則は二本立てで固定した。**①入れ替え** — 周回内で誰を救っても、死者の総数「二十七」は動かない(第4話で透が体感、第6話で田巻が言語化)。**②終わらせる** — 繰り返しそのものを止める行為は入れ替えの外にあり、別に対価を要求する(第6話で田巻が「一度だけ、別の数を見た」と漏らす伏線 → 第8話で開示)。田巻の死は保存則の結果ではなく、終わらせるために払われた代価である。
- リセット条件は「午後六時四十一分の爆発が起きた瞬間」に統一。透の生死は関係しない(第3話で確定・明示)。町を出ても戻る。
- 透を最後まで凡人のままにした。周回を重ねて万能の攻略者にせず、記憶と足で稼いだ情報しか持たせていない。
- 装置の原理は説明しきらない。第七計器室の描写は「人智を超えたまま」に留め、その分を町の日常と人物に配分した。
- 雨は毎周回、午後六時二十分前後に降り出す。透にとっては終わりの合図であり、最終話ではそれだけが残る。
