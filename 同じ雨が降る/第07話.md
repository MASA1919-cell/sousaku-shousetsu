# 第7話「第七計器室」

矢代幾造から話を聞き出すのに、透は九日かかった。

九日というのは、九回の十月十七日という意味だ。老人は毎回、同じ椅子に座って、同じ新聞を広げている。透は毎回、同じ挨拶から始めた。

一日目、矢代は「昔のことだ」と言って口を閉じた。

三日目、透は靴を持っていった。踵の減った作業靴だ。矢代は黙って受け取り、直し、代金しか受け取らなかった。

五日目、透は「あの下の、もっと下の話を聞かせてください」と言った。矢代は「誰から聞いた」と言った。透は「誰からも聞いていません」と答えた。それは本当だった。

七日目、透は自分から話した。四十七日ぶんの、誰にも言えなかったことを。老人は途中で新聞をたたんだ。信じたわけではなかったと思う。ただ、この爺さんは、人の話を最後まで聞くという習慣を持っていた。

九日目、矢代は店の奥から古い茶筒を出してきて、蓋を開け、茶ではなく錆びた釘のようなものを一本、机に置いた。

「これはな、わしが十四のときに掘った穴から出たもんだ」

　◇

昭和十九年、矢代幾造は十四歳で動員された。

潮見には海軍の工廠があり、湾を見下ろす台地の腹に、地下壕が掘られていた。矢代たちは学校ごと連れていかれて、二年近く土を運んだ。掘っているのが何なのか、誰も教えなかった。

「部屋が七つあった」矢代は言った。「計器室、と呼んどった。第一から第七まで。第一から第六までは、わしらも入った。棚と机があってな、士官が地図を広げとった。ただの部屋だ」

「第七は」

「入っとらん」

矢代は釘をつまんで、指のあいだで回した。

「第七だけは、掘ったのがわしらじゃない。よそから来た者が掘って、よそから来たものを入れた。運び込む夜は、わしらは表に出された。トラックが四台。荷は幌をかけてあった。降ろすのに一晩かかった」

「何だったんですか」

「知らん。わしは見とらん」

老人は釘を机に戻した。

「ただな、音がしとった」

透は顔を上げた。

「音?」

「壁の向こうで、こう、時計みたいな音がしとった。カチ、カチ、じゃない。もっと重い。舟の底を叩くような音でな。夜、当直で通ると、それが聞こえる。三月ごろから聞こえはじめて、だんだん間が短うなった」

「短くなった」

「早口になった、というかな。八月に入ったら、もう音と音のあいだがなかった」

矢代は茶筒の蓋を閉めた。

「そんで、八月の十五日だ。上のもんが皆、慌てて焼くもんを焼いて、埋めるもんを埋めた。第七は、入口をぜんぶ塞いだ。コンクリを流し込んでな」

「なかに、何か残ったまま?」

「一人、上がってこんかった」

店の蛍光灯が、じじ、と鳴った。

「名前は知らん。若い士官だった。それだけ覚えとる。誰も探さんかった。誰も、探すなと言われとった」

　◇

六時二十分。雨。六時四十一分。世界がめくれた。

七十三度目の朝、透は五時二十八分に起きて、六時前に家を出た。

　◇

地下三階の機械室の、いちばん奥。配電盤の裏に、床の点検口がある。

透は五十回以上この部屋に入っていたが、その蓋を開けたことは一度もなかった。開ける必要がなかったからだ。図面には「旧排水槽」と書いてある。潮見中央マートの図面はすべて、地下三階までしか描かれていない。

バールを差し込んだとき、後ろで声がした。

「やめとけ」

透は振り返らなかった。

「田巻さん、鍵、持ってますよね」

「その下に鍵はない。塞いであるだけだ」

「だったら壊します」

田巻は何も言わなかった。透がバールに体重をかけると、蓋がひとつ、鈍い音を立てて浮いた。

「相原」

「一緒に行きませんか」

透は蓋を横にずらした。下から、冷たい空気が上がってきた。地下街の匂いでも、湾の匂いでもなかった。土の匂いだった。七十年、動かなかった空気の匂いだ。

長い沈黙のあと、田巻が工具箱を置いた。

「照明は俺が持つ」

　◇

穴の下は、素掘りの通路だった。

高さは人がやっと立てるくらいで、壁に木の支保工が残っている。半分は腐って落ちていた。床に水が溜まっていて、歩くたびに膝まで冷たくなった。田巻の投光器の光が、天井の低い曲がりを照らした。

「地下街を作るときにな」田巻が前を歩きながら言った。「排水を通すのに、この壕の一部を使ったんだ。俺は市の下水道にいたから、図面の古いのを見たことがある。第一から第六まではコンクリで潰れてる。生きてるのは第七に降りる縦坑だけだ」

「知ってたんですね」

「知ってた」

通路が三度曲がって、下りになった。水が引いて、代わりに床が乾いて、砂のようになった。透は自分の足音が、遅れて返ってくるのに気づいた。壁が近いのに、返りが遅い。

「田巻さん」

「言うな。気にすると歩けん」

時計を見た。六時二分。二分前に見たときは五時五十九分だった。三分経っている。おかしい。透は歩数を数えていた。三十歩しか歩いていない。

やがて、光の先に鉄の扉が現れた。

扉は、開いていた。

押し開けられたのではなかった。中央が、内側から膨らむようにゆがんで、コンクリの充填が砕けて散っていた。何十年もかけて、なかから押されたような壊れ方だった。

「二百四十度目に、俺はここまで来た」田巻が言った。「そのときも、こうなってた」

　◇

第七計器室は、思ったより広かった。

円い部屋だった。壁は鉄板で、リベットが打ってある。船の底を切り取って埋めたような部屋だ、と透は思った。天井から太い管が何本も下りていて、床の中央のものに繋がっている。

中央にあるものを、透は何と呼べばいいのかわからなかった。

高さは人の背の倍ほど。鉄と、ガラスと、それ以外の何か。表面は錆びているのに、ある部分だけが濡れたように光っている。継ぎ目がない。鋳物なら継ぎ目がないこともある。だがこれは、鋳物が持つべき重さの感じがしなかった。

音がしていた。

舟の底を叩くような、重い音。音と音のあいだが、ない。

「触るな」田巻が言った。「触ると、見える」

「何が」

「自分が」

透は投光器を横に振った。

部屋の入口に、人が立っていた。

透は息を呑んだ。田巻ではなかった。田巻は透の左にいる。入口に立っているのは、透だった。投光器を持って、部屋を見回している透だった。それは一秒ほどそこにいて、消えた。

「何度も見た」田巻の声が平坦に戻っていた。「前に降りたときの俺だ。あるいは、これから降りる俺だ。ここは、そういう部屋だ」

透は動けなかった。

「田巻さん。これは、何なんですか」

「知らん」

「知らないって」

「知らん」田巻は言った。「これが何かを知ってた連中は、八十年前に全員死んだ。図面もない。名前もない。俺が知ってるのは、こいつが動いてるってことだけだ。三月から音がして、八月に間がなくなって、それきり誰も止めてない」

透は中央のものを見上げた。

これは、爆発の原因ではない。爆発は、上の地下街のガスと火で起きる。だが、この部屋があの日を折り返している。二十七人が死ぬ日を、八十年ぶんの錆をかぶったまま、律儀に、正確に、繰り返している。

なぜ、と聞いてはいけないような気がした。

そのとき、投光器の光の縁に、白いものが入った。

　◇

中央のものの裏側、床から一段高くなった台の上に、それはあった。

椅子があった。鉄の椅子だ。

人が座っていた。

軍服だとわかったのは、襟の形が残っていたからだ。布はほとんど土に近かった。体は、椅子の背に寄りかかることもなく、まっすぐ前を向いて座っていた。

両手が、上がっていた。

中央のものから突き出た、二本の把手を、握っていた。

透は自分の呼吸の音が部屋じゅうに響いているのを聞いた。骨の手は、把手を握ったまま、八十年そこにあった。落ちていない。ずれてもいない。まるで、いまも力をこめているように。

「……上がってこなかった一人だ」透は言った。

田巻は、その椅子を見なかった。はじめから見ないと決めているような立ち方をしていた。

「二百四十度目に、俺はここまで来た」男は言った。「そこまで見て、上がった」

「それで」

「上がって、数えた」

田巻の声が、はじめて揺れた。

「二十七じゃなかった」

「何人でした」

「わからん」田巻は言った。「わからんが、少なかった。俺が数えはじめる前に、六時四十一分になった。人が、明らかに少なかった。北の通路が、すいてた」

透は椅子のほうを見た。

握られたままの把手。八十年、離されていない手。

「あなたが、ここに来ただけで、数が減った」

「そうだ」田巻は言った。「だからもう来ない。二度と来ない。数が減るのは、俺の娘が減る側に入るかもしれんということだからだ」

男は投光器を透に押しつけて、背を向けた。

「帰るぞ。六時四十一分だ」
