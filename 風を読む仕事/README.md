# 風を読む仕事

### 人工知能が外さない時代に、あえて空を見上げる気象予報士が、山間の町で証明した「勘」の価値

## あらすじ

気象予報士の資格を取り、本庁配属を夢見ていた霧原奏(きりはら かなで)、二十五歳。だが辞令に記されていたのは、山あいの小さな地方局「山彦テレビ」だった。予報業務のほとんどを、高精度の人工知能予報システム「澄天(すみてん)」が担う時代、奏の仕事は、そのデータを読み上げるだけの、味気ないものになっていた。そんな中、局の裏山で観測小屋を営む契約観測員・猿渡岳人(さるわたり がくと)と出会う。四十年にわたり、雲の形、風の匂い、土地の癖を、自らの目と経験だけで読み続けてきた「山の目」。澄天が見逃した局地的な霧を的中させた猿渡の技術に惹かれ、奏は弟子入りを志願する。雲の名前、風の匂いの読み方――地道な修行を重ねる奏だったが、記録的な台風の接近と共に、澄天の予報と、猿渡の経験則の間に、決定的な食い違いが生まれる。二十年前、自らの判断ミスで人命を守れなかった過去を持つ猿渡と共に、奏は、人工知能と人間、どちらの目も欠かせない予報のあり方を、命がけで証明していく。近未来科学空想、全10話。

## 登場人物

- **霧原奏(きりはら かなで)** — 主人公・語り手。二十五歳の新人気象予報士。本庁配属を希望していたが、地方局「山彦テレビ」に配属される。猿渡に弟子入りし、経験に基づく天気の読み方を学んでいく
- **猿渡岳人(さるわたり がくと)** — 局の契約観測員、通称「山の目」。四十年にわたり、自らの目と経験だけで天気を読み続けてきた。二十年前、判断ミスで人命を守れなかった過去を持ち、それ以来、断定を避ける慎重な物言いを貫いている
- **桑島** — 山彦テレビ気象部のプロデューサー。人工知能予報を重視する立場だが、奏の判断力を、現実的な視点で見極めようとする
- **森下** — 気象部の若手ディレクター。奏の理解者として、様々な場面で支えてくれる

## 制作メモ

- 過去8作に続く9作目。近未来科学空想ジャンルは、月面を舞台にした第4作『静かの海郵便局』以来だが、本作は恋愛ではなく、人工知能と人間の共存という、社会的なテーマを正面から扱った
- 主人公像は「強い意志を持つ挑戦者」。これまでの作品で描いてきた「何かを失った人が再生する」「普通の人が非日常に巻き込まれる」とは異なる、目標に向かって困難を乗り越えていくタイプの主人公を、初めて主軸に据えた
- メイン軸はバトル・対立解決だが、人工知能を単純な敵として描かず、「人工知能の限界を人間の経験で補う」という共存の着地点を示すことを意識した。台風災害というクライマックスを通じて、両者の連携が結実する構成にした
- 英語の略称表記は、地の文・会話文問わず「人工知能」に統一し、英字混入ゼロの原則を貫いた。固有名詞の「澄天」のみ、システム名として使用している
