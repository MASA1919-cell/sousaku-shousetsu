# 八百年目の起動

### ガラクタと呼ばれた古代ゴーレム、少女を守って無双する

## あらすじ

大戦の終結から八百年。崩れかけた地下遺跡の奥で、一体の古代ゴーレムが目を覚ました。守護甲型八號機——避難民を守り抜くために造られた「盾」のゴーレムである。彼を起動させたのは、行方不明の両親を捜す十四歳の遺跡調べ屋見習い、ミリィ。手違いで彼女を「仮の主」として登録してしまった八號機は、錆びた体を軋ませながら、少女とともに八百年ぶりの地上へ踏み出す。鑑定額は銅貨三枚のガラクタ扱い。それでも竜を殴り、盗賊を蹴散らし、少女の涙を拭うその力は本物だった。やがて二人の前に立ちはだかるのは、八百年間戦争を終えられなかった兄弟機・九號機と、古代兵器を私する蒐集家ヴァイス卿。守るための力は、殺すための力に勝てるのか——人外の目が人の心を学んでいく、爽快冒険譚。

## 登場人物

- **八號機(ハチ)** — 主人公。守護甲型八號機。八百年前の大戦で避難民を守るために造られた「盾」のゴーレム。全高二・八メートル。語り手。一人称は「本機」。戦闘力は規格外だが、人の心の動きを「解析不能」として記録する癖がある
- **ミリィ** — 十四歳の遺跡調べ屋見習い。三年前にアルカ大遺跡で行方不明になった両親を捜している。手違いでハチの「仮の主」になった。明るく、意地っ張りで、泣き虫
- **九號機** — ハチの兄弟機。突撃型。八百年前、戦争終結の報を受け取れないまま任務を続行し続けた「終われなかった兵」。現在はヴァイス卿の外部制御器に縛られている
- **ヴァイス卿** — 蒐集家を名乗る貴族。古代兵器を発掘・収集し、戦場に売ることを画策する。私兵団「白鴉」を抱える
- **バルド** — 隊商を率いる恰幅のいい商人。ハチに命を救われて以来の理解者
- **ロッテ** — 発掘者ギルド・ラーム支部のギルド長。豪快な女丈夫。ハチの価値を最初に見抜いた一人
- **テオ爺** — 遺跡都市ヒースルの老整備師。ハチを造った賢者エルシアの工房の流れを汲む最後の職人
- **エルシア** — 八百年前にハチを造った賢者。故人。記録の中でのみ登場する

## 制作メモ

- 人外(ゴーレム)一人称の語りで、感情を「解析不能」と記録していた機械が、最終話で初めて「私」と名乗るまでの変化を縦軸にした
- 爽快枠として、各話に必ず一つ「ハチが規格外の力を見せる場面」を置いた(竜を殴る/盗賊一掃/単機強襲など)
- 敵の九號機は「悪」ではなく「終われなかった兵」として描き、倒して救う決着にした
- ミリィの両親は死なせず、東の浮遊遺跡へ向かった記録を残して「旅は続く」形の完結にした
