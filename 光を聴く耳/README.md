# 光を聴く耳

### 声のない種族と手話通訳士が結んだ、十年戦争の終わりのはじまり

## あらすじ

手話通訳士の佐倉芽衣(さくら めい)、二十八歳。ろう者の両親のもとで育ち、「音のない言葉」を読むことを仕事にしてきた。ある夜、地下鉄のホームで倒れていた謎の老人を助けようとした瞬間、光に包まれ、気づけば異世界の戦場の森に倒れていた。人間の国アルディア王国と、声を持たず肌や角の発光で会話する種族「谺人(こだまびと)」が、十年間戦い続けている境界の森。人間軍に保護された芽衣は、遠くの森で明滅する光に、手話と同じ「間」と「リズム」の文法を見出していく。斥候として出会った谺人の少女スイ、一年前に行方不明になり実は谺人に助けられ生きていた少年兵レン。二つの言葉をつなぐ小さな辞書が育っていく一方、家族を谺人に奪われたガルド将軍は、対話そのものを憎み、大規模な侵攻を決断する。声と光がぶつかる夜、芽衣を庇った隊長ヴォルフが倒れたことで、両陣営は初めて、互いの十年分の喪失と向き合うことになる。切なさを抱えながらも、前へ進もうとする者たちの、通訳の物語。全10話。

## 登場人物

- **佐倉芽衣(さくら めい)** — 主人公・語り手。二十八歳の手話通訳士。ろう者の両親を持つコーダ(聴者の子)で、「間」や「リズム」から意味を読む力に長けている。異世界の森に迷い込み、人間と谺人の通訳役となる
- **スイ** — 境ノ森に暮らす谺人の少女戦士。角の光の色が翠がかっているため、芽衣が心の中でそう呼ぶ。芽衣と少しずつ言葉を積み重ね、心を通わせていく
- **レン** — 元・第七国境守備隊の少年兵、十九歳。一年前の偵察任務で重傷を負い行方不明となるが、実は谺人に助けられ生き延びていた。谺人の光語を学び、独自の記録ノートを遺していた「通訳の芽」
- **ヴォルフ・アーレン** — アルディア王国軍第七国境守備隊の隊長。三十代半ば。芽衣を保護し、対話の可能性を最後まで信じようとする現実主義者
- **ガルド将軍** — この一帯の全軍を指揮する将軍。十年前、谺人の襲撃で妻と娘を失い、対話そのものを拒み続けてきた
- **鉄色(くろがね)** — 谺人側の戦士の頭。かつて人間軍に一族の森を焼かれ、半数近くを失った過去を持つ

## 制作メモ

- 「声を持たない種族と、手話通訳士」という組み合わせを軸にした。手話が手の形だけでなく間・速さ・表情で意味を持つように、谺人の光語も瞬きの長さとリズムで文法を持つ設定にし、芽衣の職業的な「読む力」がそのまま戦争を止める鍵になる構成にした
- メイン軸はバトル・対立の解決。第8話の夜襲でヴォルフが芽衣を庇って倒れる展開を転換点に置き、双方の指導者(ガルド将軍・鉄色)がそれぞれの十年分の喪失と向き合う場面で、対話の糸口を結んだ
- 結末は「切なさを含む・ほろ苦いが前向き」の指定どおり、ヴォルフの死という喪失を経ながらも、通訳所という形で対話が続いていく余地を残した。芽衣は元の世界に帰る道を選ぶが、それは物語の終わりではなく、彼女がこの世界に残したものが、彼女なしでも続いていくという着地にした
- 過去作と違い、主人公が異世界に留まらず元の世界へ帰還する構成は本作が初めて。現実世界とのタイムラグ(向こうでは三十秒しか経っていない)は、遅い手紙を軸にした第4作『静かの海郵便局』の時差モチーフと対をなす
