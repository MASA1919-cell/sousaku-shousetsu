# 台本のない教室

### 震えながらも、ちゃんと立てた日を積み重ねる、廃部寸前の部活と再生の物語

## あらすじ

十色すばる(といろ すばる)、高校二年生。中学時代、即興演劇(インプロ)の天才と呼ばれていたが、全国大会決勝の舞台で頭が真っ白になり、一言も発せないまま立ち尽くした。以来、「台本のない状況」が怖くなり、すべてを準備し尽くした毎日だけを、静かに送るようになった。転入した新しい学校で、名前だけ貸すつもりだった「なんでも部」――部員一人で廃部寸前の、何でも屋のような部活に、なぜか巻き込まれていく。部長の桜庭陽向(さくらば ひなた)、機械いじり一筋の因幡創(いなば そう)、人前で歌えなくなった元アイドル志望の羽鳥ののか(はとり ののか)。個性豊かな仲間たちと、学校中からの無茶振りの依頼をこなすうち、すばるは、一人では抱えきれなかった恐怖を、少しずつ手放していく。ネットに再燃した過去の動画、部長の兄が遺した部活への想い、文化祭の舞台――台本のない毎日の先に、すばるが見つけたのは、誰かと一緒だから話せる、自分だけの言葉だった。爽やかでコメディ寄りの青春群像劇、全10話。

## 登場人物

- **十色すばる(といろ すばる)** — 主人公・語り手。高校二年生の転入生。元インプロ天才児だが、大舞台でのフリーズ事件がトラウマで、台本のない状況を避けて生きてきた。なんでも部での日々を通じて、少しずつ自分の言葉を取り戻していく
- **桜庭陽向(さくらば ひなた)** — 三年生。なんでも部部長。底抜けに前向きで、困っている人を放っておけない性分。廃部寸前だった部活を一人で守ってきたが、実はそれを創設したのは、事故で亡くなった彼女の兄だった
- **因幡創(いなば そう)** — 二年生。機械いじりが得意な寡黙な部員。壊れたものを直すことに没頭する一方、自分の作品を笑われた過去を抱えている
- **羽鳥ののか(はとり ののか)** — 一年生。元アイドル志望。配信での心無いコメントがきっかけで、人前で歌えなくなった。すばると似た傷を抱える存在として、互いに支え合う

## 制作メモ

- 過去4作(異世界ファンタジー・人外バトル・ミステリ・月面科学空想)とジャンルを大きく変え、現代日本の高校を舞台にした青春群像劇に初挑戦。トーン指定「爽快・コメディ寄り」を活かし、深刻なトラウマ描写を軽妙な部活コメディの中に溶け込ませる構成にした
- メイン軸は人間関係・絆。「無茶振りの依頼を部員全員でこなす」という一話完結に近いエピソード構造を土台に、すばるの再生と、部長の兄の想いという縦軸を並走させた
- 主人公の「何かを失った人が再生する」という指定を、即興演劇のフリーズ事件による「台本のない状況への恐怖」という形に落とし込み、ラストは「震えたまま話せるようになった」という、完全な克服ではなく前進として着地させた
- 執筆時の反省を踏まえ、初稿から目標字数(3,000〜4,000字)を狙って書いたが、それでも初稿は2,000字前後に収まる傾向が変わらず、結局全話で増補が必要だった。次回はさらに長く書く意識が必要
