# 静かの海郵便局

### 月面生まれの少年と地球の少女が届ける、一.三秒より遅い言葉たち

## あらすじ

二十二世紀、月面都市・静海市(しずみし)。どんな言葉も一・三秒で地球に届く時代に、そこだけ時間の流れが違う場所がある——紙の手紙と小さな荷物だけを扱う、月でただ一つの実体郵便局「静かの海郵便局」。配達見習いの三日月湊(みかづき みなと)、十七歳。月面生まれの体は地球の重力に耐えられず、空に浮かぶ青い星を「一生行けない場所」と決めて生きてきた。そんな局に、地球からの少女・青葉ハルがアルバイトにやってくる。父の赴任に伴う、一年だけの月暮らし。亡き夫が生前に予約した「毎年届く誕生日カード」、冷凍睡眠で眠る人へ三十年後に届く手紙、太陽フレアで通信が全部止まった夜にただ一つ動いた紙の便り。一通ずつの配達が二人の距離を縮めていく——でも、ハルが地球へ帰る日は、確実に近づいていて。遅いからこそ届くものがある。ほのぼの月面ロマンス、全10話。

## 目次

1. 第1話「月で手紙を運ぶ仕事」
2. 第2話「地球見坂」
3. 第3話「六分の一の体」
4. 第4話「三十年後に着く手紙」
5. 第5話「うさぎ跳びの祭り」
6. 第6話「送信されなかった手紙」
7. 第7話「帰還通知」
8. 第8話「フレアの夜」
9. 第9話「最後の配達」
10. 第10話(最終話)「地球からの第一便」

## 登場人物

- **三日月湊(みかづき みなと)** — 主人公。十七歳の配達見習い。月面生まれ(ムーンボーン)三世で、低重力育ちの体は地球の重力に長く耐えられない。語り手。月の地図が全部頭に入っている
- **青葉ハル(あおば はる)** — 地球から来た十七歳。父の赴任に伴い一年だけ静海市に滞在、郵便局でアルバイトを始める。よく食べ、よく笑い、月の重力に三日で慣れた。地球では書道を習っていた
- **大熊タヱ(おおくま たえ)** — 静かの海郵便局の局長。七十九歳。月面入植第一世代で、局の創設者。「電気より速い言葉ばかりだと、人間の心が追いつかない」が持論
- **ポン太** — 局の旧型配達補助ロボット。六輪。言葉は話せないが、返事の電子音だけ妙に感情豊か。湊の相棒
- **青葉修平(あおば しゅうへい)** — ハルの父。月面天文台に赴任してきた太陽観測の研究者。フレア予報が専門
- **常盤サキ(ときわ さき)** — 第1話の配達先の老婦人。亡夫が生前予約した誕生日カードを、毎年受け取っている
- **氷室航(ひむろ わたる)** — 第4話の依頼人。冷凍睡眠で外惑星航路に発つ青年。三十年後の自分宛てに手紙を出す

## 制作メモ

- 「通信は一・三秒、手紙は三日」という月ならではの時差を全話の背骨にした。速い言葉と遅い言葉の対比が、そのまま二人の恋の速度になる構成
- 連作人間ドラマ(一話=一通の手紙)にロマンスの縦糸を重ねる、第3作『さよならの部屋』と同じ骨格をほのぼの側に振った
- 湊の「地球に行けない体」とハルの「月にいられない事情」を対にして、最終話は「ハルが自分の意思で月に戻ってくる」ハッピーエンドで着地
- 科学考証はやわらかめに: 月の重力六分の一、地球-月の通信遅延一・三秒、太陽フレアによる通信障害(第8話)は実在の物理に寄せた
