# 星を蒔く

### 赤い惑星に、はじめての春を

## あらすじ

酸化鉄に覆われ、赤茶けた砂ばかりが広がる開拓惑星アカホシ。二十年かけたテラフォーミングは行き詰まり、たったひとつの居住ドームには撤退の通告が届いた——帰還船が来るまで、あと九十日。着任したばかりの若き惑星生態技師・陽野実里は、枯れた試験圃場の砂の下に、赤黒い膜のような在来生命「星苔」が惑星規模で眠っていることに気づく。人類の力任せのテラフォーミングは、この星がもともと持っていた命を殺し続けていた。星を作り変えるのではなく、星と手を組む——祖母から受け継いだ「枯れた土を緑に戻す」信念だけを胸に、実里は眠る星を醒まそうとする。撤退の刻限が迫るなか、力ずくを信じる公社主任・黒瀬との対立を越えて、赤い星に緑を灯すまでの九十日を描く、爽快な惑星開拓譚。

## 登場人物

- **陽野実里（ひの みのり）** — 主人公。二十二歳の惑星生態技師（土壌・微生物担当）。地球生まれ。枯れた土地を緑に戻す活動をしていた祖母の影響で、この道を選んだ。無鉄砲だが観察眼は鋭く、一度決めたら諦めない。
- **赤石（あかいし）** — アカホシ在住二十年のベテラン探査車乗り。無愛想で口は悪いが、誰よりもこの星の地形と砂を知る男。
- **ソラ** — 八歳。アカホシで生まれ、露天の緑を一度も見たことがない子ども。実里の助手を自任する。
- **七海（ななみ）** — 実里の年上の同僚。農学者。理屈屋だが、実里の突飛な仮説に最初に耳を貸す味方。
- **黒瀬剛（くろせ ごう）** — テラフォーミング公社の惑星開発主任。力ずくで星を作り変える従来手法の信奉者で、撤退推進派。
- **榎本（えのもと）** — 居住区の区長。公社と住民のあいだで板挟みになる責任者。
- **タネ** — 実里が手製で組み上げた手のひら大の探査ドローン。土の下の生体信号を読む。

## 制作メモ

- ジャンル：宇宙・惑星開拓SF／主人公：強い意志を持つ挑戦者／トーン：爽快・前向きなハッピーエンド／全10話。
- 核となる着想は「テラフォーミング（星を作り変える）をやめ、星に眠る在来生命と共生する」という逆転の発想。既存の月面SF『静かの海郵便局』とは別方向の、本格的な惑星開拓ものを目指した。
- 撤退までの残り日数を全話に通し、爽快な推進力とした。
