# さよならの部屋

### 遺品整理士・三雲汐里が読み解く、死者たちの最後の伝言

## あらすじ

遺品整理士・三雲汐里、二十九歳。故人の部屋に残されたものを片付け、その人が最後に遺した「言葉にならなかった伝言」を家族に届けるのが彼女の仕事だ。だが汐里自身は、四年前から一つの部屋を開けられずにいる——新聞記者だった姉・遥が失踪した日のまま、時間の止まった姉の部屋を。ある日、亡くなった報道カメラマンの遺品整理で、汐里は一枚の写真を見つける。失踪直前の姉が、海辺の町の防波堤に立つ後ろ姿。姉が追っていたのは、三十年前に四人の命を奪った化学工場の火災——そして、その責任を一身に押し付けられた「設備係」は、姉妹の父だった。生前整理を望む老工員の告白、消された稟議書、現像されなかった最後のフィルム。死者たちの遺した断片を繋ぎながら、汐里は姉の失踪の夜へと近づいていく。遺品整理士が最後に整理するのは、自分自身の止まった時間かもしれない——静かな再生のミステリ、全10話。

## 登場人物

- **三雲汐里(みくも しおり)** — 主人公。二十九歳の遺品整理士。観察眼が鋭く、遺品から故人の意図を読み解く。四年前に姉が失踪して以来、姉の部屋を開けられずにいる
- **三雲遥(みくも はるか)** — 汐里の姉。地方紙の記者を経てフリーの記者に。四年前、入江町で車だけを残して失踪した。父の冤罪を晴らすため、三十年前の工場火災を追っていた
- **三雲修二(みくも しゅうじ)** — 姉妹の亡父。大鷹化学の元設備係。三十年前の火災の責任を押し付けられて職を追われ、十年前に失意のまま病死した
- **剛田健一(ごうだ けんいち)** — 汐里の勤める「こしかた整理社」の社長。元葬儀屋の大男。義理堅く、顔が広い
- **蓮見悠真(はすみ ゆうま)** — 汐里の後輩、二十三歳。軽口ばかりだが手先が器用で目端が利く
- **権田耕平(ごんだ こうへい)** — 故人。報道カメラマン。遺品の百本のフィルムが物語の扉を開く
- **蒲生辰雄(がもう たつお)** — 入江町の元工員。末期の病を得て、生前整理を汐里に「指名」してきた老人。三十年前の火災の生き証人
- **如月早苗(きさらぎ さなえ)** — 大鷹化学会長の元秘書。内部告発者として遥と接触していた
- **大鷹宗吾(おおたか そうご)** — 大鷹化学会長。三十年前の火災当時の工場長。記録の改竄を主導した
- **大鷹亮介(おおたか りょうすけ)** — 現社長。宗吾の甥。四年前のあの夜、防波堤にいたもう一人
- **田尻靜枝(たじり しずえ)** — 入江町の民宿の女将。遥の滞在先だった

## 制作メモ

- 連作ミステリの形式: 前半は一話完結の遺品整理ミステリ(止まった時計/開かずの金庫)で主人公の流儀を確立し、第3話から姉の失踪という縦糸に合流させた
- 手がかりはすべて「遺品」で統一(写真・ノート・録音テープ・稟議書・未現像フィルム)。第3話でさりげなく登場した未現像フィルムが第9話の決定的証拠になる伏線構造
- シリアス指定に合わせ、真相は重く(姉は事故ではなく事件死)、ただし読後感は「四年間開けられなかった姉の部屋を自分の手で整理する」再生で締めた
- 冤罪・内部告発・企業城下町という社会派要素は、あくまで「家族の物語」の背景に留める配分にした
