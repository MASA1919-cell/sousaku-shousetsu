## しくみ

- `index.html` — ビューア(単一ファイル・外部依存なし)
- `data.js` — ビューアが読む全作品データ(`build.js` が各フォルダの .md から生成)
- `build.js` — 原稿(.md)から `data.js` を作る Node スクリプト

ローカルで読む場合は、このリポジトリをダウンロードして `index.html` をブラウザで開くだけです。

収録作品はすべてオリジナル小説です。著作権は作者に帰属します。無断転載・複製はご遠慮ください。
