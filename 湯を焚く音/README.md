# 湯を焚く音

### 祖父の銭湯を継いだ三代目が、常連客との一年間で見つけた、人と人をつなぐ湯加減

## あらすじ

東京でイラストレーターを目指して七年、芽が出ないまま派遣の仕事を続けていた時任結愛(ときとう ゆあ)、二十六歳。祖父・久蔵の訃報を受けて帰郷した彼女を待っていたのは、廃業寸前の小さな銭湯「久乃湯(ひさのゆ)」だった。祖父の古い友人・米田徳治(よねだ とくじ)に助けられながら番台に座り始めた結愛は、常連客たちとの交流を通じて、祖父がこの場所で守ってきたものの重さを、少しずつ知っていく。足腰の悪い柳瀬さんの心の拠り所、シングルマザーの早坂親子の生活の支え、閉鎖の決まった町工場に勤める南条要の疲れを癒す居場所。土地の売却を持ちかける不動産会社の提案に揺れながらも、結愛は銭湯を継ぐ決意を固め、色褪せた富士山のペンキ絵を、自分の手で塗り直す。街の人々の力を借りながら、彼女は、祖父の遺志と、自分自身が諦めかけていた夢の両方を、静かに取り戻していく。温かなヒューマンドラマ、全10話。

## 登場人物

- **時任結愛(ときとう ゆあ)** — 主人公・語り手。二十六歳。東京でイラストレーターを目指すも芽が出ず、祖父の訃報を機に帰郷し、銭湯「久乃湯」を継ぐことになる
- **米田徳治(よねだ とくじ)** — 祖父・久蔵の古い友人。結愛に銭湯の仕事を一から教える指導役。物語終盤、番台の向こう側から、一人の客へと居場所を移していく
- **早坂美咲(はやさか みさき)** — シングルマザーの常連客。息子のそらと共に、久乃湯を生活の一部にしている。久蔵に受けた恩を、結愛にも返そうとする
- **南条要(なんじょう かなめ)** — 無口な町工場の溶接工。工場の閉鎖を控えながらも、久乃湯の修繕に力を貸してくれる
- **柳瀬トメ** — 足腰の悪い老婦人。久蔵が特別にしつらえた手すりを頼りに、毎日湯船に浸かりに来る
- **桐生** — 不動産会社の社員。当初は土地の売却を提案するが、結愛の決意を見て、クラウドファンディングの協力者へと転じる

## 制作メモ

- 過去7作(異世界ファンタジー・人外バトル・ミステリ・月面科学空想・現代異世界ファンタジー・青春群像劇・ホラー怪異)に続く8作目。現代日本の大人のヒューマンドラマは本作が初で、「何かを失った人が再生する」主人公像を、家業の継承と地域コミュニティという題材で描いた
- メイン軸は人間関係・絆。一話ごとに異なる常連客との関わりを積み重ね、最終的に街ぐるみで銭湯を守る展開へ収束させる、連作の構成にした
- 結末は静かで温かいハッピーエンド。主人公が諦めていた「絵を描くこと」を、銭湯の再生と重ね合わせることで、家業の継承と自己実現を同時に着地させた
- 執筆時の反省を踏まえ、初稿の時点で短くなりがちな傾向に対し、各話とも執筆直後に増補する方式を継続したが、全体字数はやや控えめ(約22,200字)。会話とエピソード単位の切り替えが多い構成のため、地の文の厚みが他作品よりやや薄くなった
