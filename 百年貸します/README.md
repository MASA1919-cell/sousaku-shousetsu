# 百年貸します

### あわい町の損料屋・九十九屋人情帖

## あらすじ

人の世とあやかしの世の、ちょうど間にある「あわい町」。その路地の奥に、鍋釜から婚礼道具まで何でも貸す損料屋「九十九屋」がある。四代目の若旦那・千吉は、昼行灯と評判の飄々とした男だが、物に宿った想いの声を聴き、客と貸し物を見立てる目は決して外さない。損料は銭でなくてもいい。物語ひとつ、大根一本、ふた月後の飯ひと椀。人にも、あやかしにも、誰にでも貸すのが店の流儀。そして帳面に記された貸しは、返ってくるまで決して消えない——たとえ百年経っても。傘一本、鍋ひとつをめぐる小さな人情の連作の先に、帳面の最後に残る百年前の貸し出し「琴 月草 嵩山の主さま」の真心が明かされる、温かな和風あやかし人情譚。

## 登場人物

- **千吉（せんきち）** — 主人公。損料屋「九十九屋」の四代目。二十代半ばの飄々とした若旦那。物に宿った想いの声を聴ける。口癖は「うちは、誰にでもお貸しします」。
- **お銀（おぎん）** — 九十九屋の番頭。化け猫。姉御肌でそろばん達者。昔、人間の商家に飼われていた。
- **豆助（まめすけ）** — 九十九屋の小僧。豆狸。化けるのが下手で、しっぽがよく出る。
- **お絹（おきぬ）** — 長屋に越してきた母。息子の太一とふたり暮らし。形見の琴爪を持つ。
- **太一（たいち）** — お絹の息子。七歳。
- **嵩山の主（かさやまのぬし）** — 町はずれの嵩山に棲む白い大狼。百年、山を降りていない。
- **小夜（さよ）** — 百年前、麓の村にいた琴弾きの娘。琴「月草」のもとの持ち主。
- **火吉（ひきち）** — 長屋に住むひょっとこの妖。火の扱いの名人。

## 制作メモ

- ジャンル：和風ファンタジー×妖怪（本プロジェクト初）／主人公：飄々とした達人・変わり者（初のタイプ）／トーン：ほのぼの・温かい人情もの／全10話。
- 一話＝一つの貸し物と一組の客の連作人情話。縦軸は「帳面から百年消えない貸し出し」で、第7話から本格化して最終話で回収する。
- 「損料屋」は江戸に実在した貸し物商売。物は長く使われ想いが宿ると付喪神になる＝店名「九十九屋」の由来。
