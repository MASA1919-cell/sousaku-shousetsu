# 追放された定年魔道具師のネジ巻きスローライフ
### 〜「時代遅れ」と言われた俺の手仕事、実は王都の生命線でした〜

全10話完結のオリジナル異世界転生小説。1話あたり約4,000〜4,500字(読了目安 約10分)。

## あらすじ

町工場ひと筋50年の旋盤職人・真崎源三(68)は、旋盤の前で人生を終え、異世界の宮廷魔道具工房の掃除係「ゲン」として目を覚ます。転生特典は、触れた物の寸法・歪み・魔力の流れが視える『マイクロメーターの眼』——だが本当のチートは、五十年かけて体に入れた職人の腕だった。

誰にも知られず王都の魔導設備を毎朝こっそり調整していたゲンは、新長官に「時代遅れ」と追放される。辺境のオルド村で念願の修理工房「げんこつ工房」を開き、のんびり暮らし始めるが——彼の「朝の散歩」を失った王都のインフラは、静かに軋みはじめていた。

## 登場人物

- **ゲン(真崎源三)** — 主人公。68歳。前世は東京下町の旋盤職人。転生特典『マイクロメーターの眼』持ち。流儀は「機械の声を聞け」
- **フィオナ(フィオ)** — 16歳。元・宮廷工房の見習い掃除係。ゲンの一番弟子。直感と手で覚えるタイプ
- **クルト・アレン** — 17歳。王立学院首席の天才魔道具師。理論派。挫折を経てゲンに弟子入り
- **テツ** — 鉄くずを食べる三尾の狸型魔獣。金属疲労を嗅ぎ当てる、げんこつ工房の検査主任
- **バルデル・モーガン子爵** — 宮廷工房の新長官。「自動魔力成形」の大家。ゲンを追放した張本人だが、悪人ではなく視野の狭い技術者→後に改心
- **アリシア・アルデン** — 第二王女、19歳。「計算姫」。現場主義の監察役で、ゲンの功績を最初に見抜く側
- **マーサ** — オルド村の宿屋「山鳩亭」の女将。口は悪いが飯は絶品
- **ドラン** — 行商人。ゲンをオルド村に導き、後に中古魔道具商組合長
- **ハンス爺** — オルド村の村長

## 制作メモ(2026年7月時点のトレンド反映)

2025〜2026年の異世界転生トレンドをリサーチして設計:

- **追放×スローライフ**のハイブリッド(現在の売れ筋構成。単純な復讐劇より「追放→新天地で無自覚に大活躍→元の職場が困る」型)
- **ざまぁの質的向上**: 敵役モーガン子爵を破滅させず「ヤスリがけからやり直し」で救済する、後味のよい現代型ざまぁ
- **主人公の意外性**: 若者×剣と魔法ではなく、68歳の定年職人。チートの実体は魔法ではなく「保全・公差・段取りという製造業の知恵」
- **もふもふ枠**: 鉄くず食い魔獣テツ
- **ヒロイン枠を恋愛でなく師弟関係に**: シニア主人公に合わせ、「技術の継承」を感情の軸に
- テーマ: 「作る技術」だけでなく「保たせる技術」への敬意。ラストの定義——スローライフとは"手入れのいい人生"のこと
